Ideas Principales

  • Las funciones son uno de los pilares de js.
  • Una función es un procedimiento en Js - un conjunto de conjunto de sentencias que realizan una tarea o calculan un valor
  • Para usar una función debe declarar en algún lugar y se ejecuta cuando algo la llama

Resumen

Tipos de datos


  • Cadena (string)


    Un valor de cadena esta formado por una cadena de cero o mas caracteres Unicode (letras, difitos y signos de puntuacion). El tipo de datos string se usa para representar texto en JavaScript.

  • Numeros boleano


    Mientras que los tipos de datos de cadena y numero pueden tener un numero prácticamente ilimitado de valores diferentes, el tipo de datos booleano solo puede tener dos.
